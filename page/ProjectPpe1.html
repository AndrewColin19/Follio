PPE4 SLAM PROJET Médecins Partie 2 : consultation des médecins dans un mobile android

contexte

Le laboratoire Galaxy Swiss Bourdin (GSB) est amené dans ses différentes activités à contacter les médecins installés, par exemple les visiteurs médicaux du laboratoire se déplacent afin de présenter les nouveaux produits pharmaceutiques. Une base de données de médecins est utilisée dans ces différents processus. Cette base de données évolue fréquemment, ceci à cause de départ à la retraite de médecins ou d’installation de nouveaux praticiens. Les applications doivent intégrer ces évolutions des données. Il a été décidé de confier à une société de services informatiques la responsabilité de développer deux applications web donnant accès aux informations de la base de données.

Documents

Un certain nombre de contraintes ont été fixées :
- Le développement sera réalisé en Java. Le système d'exploitation cible sera android 4 (API 15).
- Dans un premier temps, l'application utilisera un service Web de test fourni et décrit ci dessous.
- Dans un deuxième temps, vous créerez le service Web utilisant la base de données des médecins : il sera intégré à l’application back office réalisé en PHP Zend.
- Les informations exposées indiqueront simplement les nom, prénom, adresse, téléphone et spécialité médicale complémentaire des médecins généralistes

Utilisation

Dès son lancement, l'application affiche une liste de départements.
Un appui sur un des numéros de département déclenche l'affichage de la liste des médecins du département.
Dans cette phase de prototypage, le design est libre

Ressources

Le service Web Le service Web de test est à l'url : http://denis.brodard.free.fr
Il propose les deux requêtes suivantes :
A – liste des départements :
http://denis.brodard.free.fr/dep.php qui retourne un flux XML de ce format :

                            <Departements>
                                <Departement>
                                    <num>75</num>
                                </Departement>
                                <Departement>
                                    <num>95</num>
                                </Departement>
                            </Departements>

B – liste des médecins (par exemple pour le département 75):
http://denis.brodard.free.fr/med.php?loic=1&dep=75
qui retourne un flux XML de ce format :

                            <Medecins>
                                <Medecin>
                                    <nom>Daminel</nom>
                                    <prenom>Elise</prenom>
                                    <adresse>12 rue primer 75012 Paris</adresse>
                                    <specialite>Rhumato</specialite>
                                    <tel>01 66 70 36 06</tel>
                                </Medecin>
                                <Medecin>
                                    <nom>Dupont</nom>
                                    <prenom>Alain</prenom>
                                    <adresse>12 rue taine 75012 Paris</adresse>
                                    <specialite/>
                                    <tel>01 43 70 36 06</tel>
                                </Medecin>
                            </Medecins>
